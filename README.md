# Human Activity Recognition Dissertation

## MSc Software Engineering

**Title**

A Comparative Study of Transfer Learning-Based CNN Architectures for Image-Based Human Activity Recognition

**Author**

Ujjwal Shrestha

---

This repository contains the implementation for the MSc dissertation project, including:

- Dataset analysis
- Data preprocessing
- Transfer learning using ResNet and MobileNet
- Model training
- Performance evaluation
- Comparative analysis

## Prerequisites

- Python 3.11.x
- VS Code (recommended)
- Git

---

## Clone the Repository

git clone <repository-url>

cd HAR_Dissertation

---

## Create a Virtual Environment

python3.11 -m venv .venv

---

## Activate the Virtual Environment

### macOS / Linux

source .venv/bin/activate

### Windows

.venv\Scripts\activate

---

## Install Dependencies

pip install -r requirements.txt

---

## Verify Installation

python --version

Expected:

Python 3.11.x
